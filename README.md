# 🎯 코웨이 영업 마스터 트레이닝

코웨이 홈케어닥터를 위한 렌탈 영업 트레이닝 플랫폼입니다.
매트리스케어 서비스부터 렌탈 영업까지, 최고의 영업 전문가로 성장하세요!

## ✨ 주요 기능

### 1. 🎭 고객 시나리오 시뮬레이터
- 5가지 고객 유형별 대화 연습
- AI 기반 고객 응답 시뮬레이션
- 실시간 영업 팁 제공

### 2. 📝 영업 스크립트 라이브러리
- 상황별 효과적인 멘트 모음
- 거절 대응 화법
- 성공 사례 공유
- 카테고리별 검색 기능

### 3. 🧠 제품 지식 퀴즈
- 10개 이상의 실전 퀴즈
- 즉시 피드백 및 설명
- 점수 및 정답률 추적
- 답안 리뷰 기능

### 4. 📊 실적 추적 & 목표 관리
- 일/주/월 실적 기록
- 목표 달성률 시각화
- 영업 기록 관리
- 진행 상태 추적

### 5. 🎤 롤플레잉 연습
- 5가지 실전 시나리오
- 베테랑 영업사원 모범 답변
- 연습 기록 및 피드백
- 점수 평가 시스템

### 6. 👥 커뮤니티
- 동료들과 노하우 공유
- 질문/답변 게시판
- 이달의 베스트 영업사원 랭킹
- 성공 사례 공유

## 🚀 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

브라우저에서 `http://localhost:5173` 접속

### 빌드

```bash
npm run build
```

## 🛠️ 기술 스택

- React Router 6
- Vite
- CSS3
